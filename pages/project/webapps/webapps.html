Web applications

Ajoute ton grain de sel is a web application built in javascript and node.js. It was deployed using Google App Engine and IBM's Cloudant as a database, but is now depricated. The open ended app invited users to share the food that they were eating with other users. With a title, description and optional recipe, ajoutetongraindesel.com aimed at connecting and inspiring users around food. The app was designed during the 2020 pandemic, at a time when eating alone was the norm for some people, it tried to re-introduce the communial elements surrounding eating into people's life.

Badminton team builder was built in html, css, javascript, php and SQLite and is used to manage the making of teams in badminton events where players level and number as well as available number of courts can vary. Players are stored in a database so they don't have to be registered every time and the website is designed to be used by multiple event manager without conflict using a login and registration system. There are different algorithms for team creation as well as options to prevent redundancy between teams.
